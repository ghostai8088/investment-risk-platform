## North star
A **full-scope enterprise investment-risk platform** â€” multi-tenant, auditable, reproducible, governed â€”
**not an MVP or POC**. Every slice is built to enterprise standards from the first commit: tenant isolation
(RLS), co-transactional fail-closed audit with a hash chain, lineage on every governed write, selective
bitemporality, deny-by-default entitlements, and CI that enforces the ratified standards. Scope is delivered
in thin, reviewed, independently green slices â€” depth and governance are never traded for speed.

## Delivery method (every slice)
UltraCode planning workflow â†’ committed plan doc â†’ implementation â†’ multi-lens adversarial review â†’ fix
in-scope findings â†’ `make check` â†’ **commit only on explicit approval** â†’ watch CI to green. See
`claude_operating_instructions.md`.
