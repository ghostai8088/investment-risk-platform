# Enterprise Review Prompt

Review the current design or implementation as if you are an enterprise asset management technology, risk, and control reviewer.

Assess the work across the following dimensions:

1. Architecture weaknesses
2. Product scope gaps
3. Data model gaps
4. Public asset coverage gaps
5. Private asset coverage gaps
6. Market risk methodology concerns
7. Credit risk methodology concerns
8. Counterparty risk methodology concerns
9. Liquidity risk methodology concerns
10. Scenario and stress testing gaps
11. Limit monitoring gaps
12. Breach workflow gaps
13. First Line of Defense workflow gaps
14. Second Line of Defense workflow gaps
15. Model governance gaps
16. Data lineage gaps
17. Auditability gaps
18. Security and entitlement gaps
19. Testing gaps
20. Documentation gaps
21. Enterprise buyer objections
22. Production hardening gaps

Return:
- Critical issues
- High-priority issues
- Medium-priority issues
- Low-priority issues
- Recommended remediation backlog
- Questions requiring human judgment
